In mathematics, $f(x)$ means "the value of function $f$ when the input is $x$". In a context where many variables are known it exist, it is implied that $f$ is a function of $x$, but not a function of other variables.

Example of evaluating a function

If $f(x) = x^2 + 1$, then:

  • $f(2) = 2^2 + 1 = 5$
  • $f(5) = 5^2 + 1 = 26$

Functions of multiple variables

Functions can also take multiple inputs. For example, $g(x, y) = x + y$ means that the output is the sum of the two inputs.

For example, $g(3, 4) = 3 + 4 = 7$.
